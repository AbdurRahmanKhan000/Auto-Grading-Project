# Auto Grading LMS ????`n`nAn automated code evaluation and Learning Management System (LMS) built to streamline programming education. This platform allows instructors to manage coding assignments and provides students with an interactive environment to practice, submit code, and receive immediate, automated feedback.`n`n## ?? Core Features`n`n* **Automated Code Execution:** Securely compiles and evaluates student code submissions against predefined test cases (`CodeExecutionService`).`n* **Adaptive Learning Engine:** Dynamically adjusts to student performance to provide tailored practice problems (`AdaptiveEngineService`).`n* **Role-Based Access Control:** Secure login and registration systems with distinct dashboards for Instructors and Students.`n* **Instructor Dashboard:** A centralized hub for educators to create assignments, define test parameters, and monitor student progress.`n* **Interactive Practice Arena:** A built-in web interface for students to write, test, and submit code directly in the browser.`n`n## ??? Technology Stack`n`n* **Backend:** Java, Spring Boot`n* **Build Tool:** Maven`n* **Database:** MySQL (Configured via `application.properties` and `data.sql`)`n* **Frontend Template Engine:** HTML5, CSS3 (Thymeleaf/Static Web Templates)`n* **Version Control:** Git & GitHub`n`n## ??
